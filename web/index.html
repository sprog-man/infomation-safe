Crypto Playground

            
独立操作每个密码学组件，快速验证算法行为。


            

                
📖 操作步骤

                

                    

                        1
                        

                            生成密钥
                            
在 "RSA Key Generation" 卡片中，选择密钥长度（512/1024/2048），点击 Generate Keypair。公钥和私钥会显示在下方输入框中。

                        

                    

                    

                        2
                        

                            AES 加密
                            
在 "AES-128 Encrypt/Decrypt" 卡片中输入任意文本，点击 Encrypt。密文会以 hex 形式显示在下方输入框中。

                        

                    

                    

                        3
                        

                            AES 解密
                            
保持上一步的密文不变，点击 Decrypt。结果会显示在 Result 区域，应与原文一致。

                        

                    

                    

                        4
                        

                            HMAC 认证
                            
在 "HMAC-SHA256" 卡片中，先用 Key 和 Message 计算签名 (Compute)，然后用相同参数验证 (Verify)。也可以故意改一个字符测试验证失败。

                        

                    

                    

                        5
                        

                            快速验证
                            
"Quick Roundtrip" 卡片一键完成 AES 加密→解密，验证原文与解密结果是否匹配。

🔑 密钥交换过程

                    

                        

                            

                                
发送端 (Client)

                                
    
                                    
  1. 生成 16 字节随机 AES 会话密钥
    2. 
                                    
  2. AES-128 加密传感器数据 → 密文
    3. 
                                    
  3. RSA 公钥加密 AES 密钥 → 256 字节
    4. 
                                    
  4. HMAC-SHA256 计算密文签名
    5. 
                                    
  5. 打包发送: [RSA密文|HMAC长度|HMAC|密文]
    6. 
                                

                            

                            
───── TCP ─────> 

                            

                                
接收端 (Server)

                                
    
                                    
  1. 接收 TCP 帧
    2. 
                                    
  2. RSA 私钥解密 → 恢复 16 字节 AES 密钥
    3. 
                                    
  3. HMAC-SHA256 验证签名
    4. 
                                    
  4. AES-128 解密密文 → 原始传感器数据
    5. 
                                    
  5. 发送 "ACCEPT"
    6.
